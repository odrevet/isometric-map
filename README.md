# Python version 

This documentation assumes that the `python` command is binded with `python 3.9` or better.

# Dependencies 

* pygame
* pygame_gui
* pyganim
* parse

```
pip install pygame pygame_gui pyganim parse
```

# Run 

* Game 
```
python main.py 
```

* Game without on screen debug

```
python -O main.py 
```

* Editor

```
python editor.py
```
